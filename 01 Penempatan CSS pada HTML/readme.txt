Ada beberapa cara untuk menempatkan css di dokumen html, yaitu:
|+| Embed
    Menempatkan css di bagian tag <head> </head>.
|+| Inline
    Menempatkan css langsung pada bagian yg ingin diubah, misalnya menempatkan css pada tag <p> </p>
|+| External
    Membuat file berekstensi (.css) dan kemudian menghubungkan antara file html dan css dengan cara menambahkan
    syntax berikut:  <link rel="stylesheet" type="text/css" href="nama_file_css_nya.css"> kedalam tag <head> </head>.
